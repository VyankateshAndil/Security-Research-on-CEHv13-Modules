# 🧠 Key Concepts Covered

## 1️⃣ Ethical Hacking

Ethical hacking is the process of legally testing computer systems, applications, and networks to identify security vulnerabilities before malicious attackers exploit them.

Ethical hackers use the same methodologies and tools as attackers, but operate with proper authorization to improve an organization's security posture.

---

## 2️⃣ Types of Hackers

| Hacker Type             | Description                                                        |
| ----------------------- | ------------------------------------------------------------------ |
| White Hat               | Authorized security professionals performing ethical hacking       |
| Black Hat               | Malicious attackers exploiting systems illegally                   |
| Gray Hat                | Hackers who may violate rules but without malicious intent         |
| Script Kiddies          | Individuals using pre-built tools with limited technical knowledge |
| Hacktivists             | Hackers motivated by political or social causes                    |
| State Sponsored Hackers | Government-backed cyber attackers                                  |
| Cyber Terrorists        | Attackers targeting critical infrastructure                        |

---

## 3️⃣ Information Security Principles

The core principles of Information Security are known as the **CIA Triad**.

### Confidentiality

Protecting sensitive information from unauthorized access.

### Integrity

Ensuring information remains accurate and unaltered.

### Availability

Ensuring systems and information remain accessible whenever required.

---

## 4️⃣ Ethical Hacking Phases

Ethical hacking follows a structured methodology.

### Reconnaissance

Gathering information about the target.

### Scanning

Identifying open ports, running services, and vulnerabilities.

### Gaining Access

Exploiting discovered vulnerabilities.

### Maintaining Access

Establishing persistence within the compromised system.

### Covering Tracks

Removing evidence of the attack to avoid detection.

---

## 5️⃣ Cyber Kill Chain

The Cyber Kill Chain describes the lifecycle of a cyber attack.

1. Reconnaissance
2. Weaponization
3. Delivery
4. Exploitation
5. Installation
6. Command & Control
7. Actions on Objectives

Understanding these phases helps defenders detect and stop attacks at different stages.

---

## 6️⃣ Information Warfare

Information Warfare involves using information and communication technologies to gain strategic advantages over adversaries.

Examples include:

* Cyber Espionage
* Psychological Operations
* Electronic Warfare
* Cyber Attacks against Critical Infrastructure
