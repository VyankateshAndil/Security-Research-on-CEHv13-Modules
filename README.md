# CEH Practical Report – Ethical Hacking & Cybersecurity

## 📌 Project Overview

This repository contains a comprehensive technical report based on the **Certified Ethical Hacker (CEH v13)** curriculum.

The project focuses on the hands-on implementation of ethical hacking tools, penetration testing methodologies, vulnerability assessment techniques, and cybersecurity concepts supported with detailed **Proof of Concepts (PoCs)**.

All experiments were conducted in controlled laboratory environments strictly for educational and ethical learning purposes.
